# FakeGPS 虚拟定位 App

基于 Kotlin + Jetpack Compose 的安卓虚拟 GPS 定位工具。

## 功能
- ✅ 导入 KML 文件（含 `<Point>` 坐标）
- ✅ 列出所有坐标点，**可勾选多个**
- ✅ **巡航模式**：勾选多个坐标→从第一个到最后一个依次模拟
- ✅ 每个坐标停留 **5~25 分钟随机**（可滑动调节）
- ✅ 前台服务 + 通知栏实时状态
- ✅ 悬浮窗快捷查看

## 使用方法

### 编译 APK（方法一：Android Studio）
1. 在 Android Studio 中打开此文件夹
2. 连接手机 → Run

### 编译 APK（方法二：GitHub 云编译，推荐）
1. 把项目推到 GitHub 仓库
2. 进入仓库 → **Actions** → **编译APK** → **Run workflow**
3. 等 3~5 分钟 → 下载 APK

### 安卓端配置（首次使用）
1. 设置 → 开发者选项 → 选择模拟位置应用 → 选"虚拟GPS"
2. 打开 App → 左下角 + 导入 KML
3. **单点模拟**：点击任意一行
4. **巡航模拟**：勾选多个点 → 点底部"巡航"按钮

## 巡航模式说明
- 按勾选顺序从第一个走到最后一个
- 每个点停留时间在设置的区间内随机
- 通知栏显示 `巡航中: 2/5` 和当前坐标名
- 最后一个点驻留直到手动停止
- 可随时点底部"停止"按钮

## KML 格式要求
```xml
<Placemark>
    <name>N1（示例塔）</name>
    <Point>
        <coordinates>经度,纬度,高度</coordinates>
    </Point>
</Placemark>
```
